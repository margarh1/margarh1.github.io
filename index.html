Homework #1 - Javascript Basics

Four functions written to practice the Javascript basics: one sings all 99 verses of "99 Bottles of Beer on the Wall", another returns an array of prime numbers, a different one generates Shakespearian insults, and the last determines whether a given string is a palindrome.

Homework #3 - Tic-Tac-Toe

A tic-tac-toe board for two players. The pieces can be colored one of seven basic colors and can be changed with the simple click of a button. The pieces themselves can likewise be swapped without needing to reload the page, though this is only possible between games. The scoreboard will still be able to track wins and losses, regardless of changes to the player pieces and the piece colors.
